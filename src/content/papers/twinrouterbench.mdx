A step-level benchmark for choosing the cheapest sufficient model tier at each LLM call in a multi-step agent, conditioned only on the prefix visible before that call. The fast static track exposes 970 router-visible prefixes from 520 instances across SWE-bench, BFCL, mtRAG, QMSum, and PinchBench, with execution-verified tier labels and deterministic scoring over label correctness, trajectory membership, and token cost. The live dynamic track runs routers end-to-end on SWE-bench Verified and reports official resolution alongside realized API spend, so offline router gains can be tested without confirming them with a live harness.
