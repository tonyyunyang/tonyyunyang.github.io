§ Colophon

A note on the typography, the palette, and the tools.

This page is here because every printed plate names its inks. The same spirit applies on the web.

I. Type

Three faces, plus a calligraphic Chinese companion. The display carries the prose, the sans carries the deks, and the mono carries the marginalia (numbers, captions, dates, codes).

  • EB Garamond

    display · 400 + 500, italic + upright

    Google Fonts (free)

  • Inter

    body sans · 400 / 500 / 600 + 400 italic

    Google Fonts (free)

  • JetBrains Mono

    captions, codes, dates · 400 / 500

    JetBrains (free)

  • LXGW WenKai

    Chinese display · 500

    霞鹜文楷, OFL (free)

II. Palette

Seven tokens, no eighth. Paper and ink make the page. Emerald carries every link. Featured cream sits behind the things you most want a visitor to see.

  • paper #F5EFE2 primary background
  • paper-shade #EDE6D5 card + plate fill
  • ink #0F1417 primary text
  • ink-soft #4A5159 secondary text + meta
  • accent #0E5347 links + highlights
  • hairline #D9D2C2 borders + dividers
  • featured #FAF1D8 featured tints

III. Tools

Fewer than you'd expect. No React, no SSR, no design framework, no template ancestry. The site is hand-set in Astro and ships through GitHub Actions.

  • Astro 5 static site framework, MDX content collections
  • Tailwind v4 design tokens via @theme block
  • Pagefind client-side fuzzy search powering ⌘K
  • Playwright visual checkpoints + the OG render helper
  • GitHub Pages deploy via Actions on every push to main

IV. Motion

Three durations, three shapes. Tap is for direct feedback (160 ms, ease‑in‑out). Warm is for state shifts (280 ms, cubic ease). Reveal is for entries (600 ms, cinematic). All of it honors prefers-reduced-motion.

V. Author

Designed and built by Tony Yang in Amsterdam, between papers. Source is on GitHub. The studio is here.
